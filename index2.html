Machine Learning Project --Otto Group Product Classification Challenge

This project is from the competition of Kaggle, our purpose is building the predictive model to distinguish between main product categories of Otto Group products. Firstly, we are using Gradient boosting trees, Gradient boosting logistic regression, Neural network, Random forest and linear discriminant analysis to build stacking models in order to classify a dataset with 93 features for more than 200,000 products. Secondly, we use folds for train dataset to predict the stacking models. Thirdly, we use calibration and model ensemble technique to minimize the logloss. Finally, our lightweight model achieved a logloss of 0.42593, and our team’s rank is in top 8% for this Kaggle competition.
