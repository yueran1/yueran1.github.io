Machine Learning Project --Otto Group Product Classification Challenge

This project is from the competition of Kaggle, our purpose is building the predictive model to distinguish between main product categories of Otto Group products. Firstly, we are using Gradient Boosting Trees, Gradient Boosting Logistic Regression, Neural Network and Random Forest to build stacking models in order to classify a dataset with 93 features for more than 200,000 products.
